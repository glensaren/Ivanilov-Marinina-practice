Game Master Helper - инструмент по созданию и редактированию карт НРИ

Цели

Цель проекта – разработать инструмент по созданию и редактированию карт НРИ, который нивелирует все минусы других сервисов. В частности, сделать “Единое ядро” всех уровней проработки карт для более удобной настройки карты. Также предполагается наличие процедурной генерации карт и детальной ручной настройки, включающая в себя инструментарий и множество ассетов.

Задачи и Этапы реализации

  1. Создание веб-сайта, на котором будет осуществляться создание карт. 
  2. Создание понятного интерфейса веб-сайта.
  3. Создание инструментария в редактора карт на веб-сайте.
  4. Создание программного обеспечения редактора для размещения его на сервере.
  5. Настройка сервера для работы с редактором и веб-сайтом.
  6. Первичная настройка базы данных на сервере.
  7. Продумывание экономической модели для дальнейшей монетизации проекта.
